## 四、底線（不能再低）

```
建置費底線：$4,000
├── 低於這個你虧時間
├── 假設你花 100 小時（保守）
├── $4,000 / 100hr = $40/hr
├── 加拿大 junior developer 時薪 $30-40
├── 你做的是 full-stack + AI → 應該是 $60-100/hr
└── $4,000 已經是友情價

月維護底線：$199
├── 實際成本：Vercel $20 + Supabase $25 + 你的時間
├── 每月大概花 2-3 小時維護 = $66-100 人力成本
├── $199 - $45（基礎設施）- $80（人力）= $74 利潤
└── 低於 $199 不值得

會員系統底線：$99 不打折
├── 這是 SaaS 產品，統一定價
├── 如果給她 $49 → 未來每個客戶都要 $49
├── SMS 成本 $3 + API $2 = $5/月 → $99 很合理
└── 永遠不打折
```

## 五、如果她說「太貴了」

### 回應策略

```
第一步：問她的預算
  「你覺得合理的範圍是多少？」
  → 她說一個數字 → 你知道差距多大

第二步：如果差距 < 20%（她說 $250/月）
  「我可以把月維護調到 $229，
    前 3 個月會員系統免費試用。」
  → 小讓步 + 試用 = 降低壓力

第三步：如果差距 > 30%（她說 $150/月）
  「這個價格我可以做基礎官網，
    但電商功能要在 Phase 2 加。
    你要不要先從官網開始？」
  → 不降價，降功能

第四步：如果她只想要 Shopify 的價格
  「Shopify 確實月費比較低，
    但你還需要另外買 Loyalty App、Email App、
    而且沒有堂數管理。
    你算一下全部加起來其實差不多。
    但如果你現在預算真的有限，
    Shopify 也是一個好的起點。」
  → 不硬賣，讓她自己選
  → 如果她最後選 Shopify，你還是可以賣她會員系統 $99/月
```

### 絕對不做的事

```
❌ 不要降到 $199/月以下（維護費）
❌ 不要會員系統打折（$99 是底線）
❌ 不要免費做網站說「之後月費賺回來」（很多 freelancer 栽在這裡）
❌ 不要承諾「不限次數修改」（會被無限改死）
❌ 不要因為她是第一個客戶就不收錢
```

## 六、定價心理學

```
為什麼建置費用「加購」結構而不是「總價」：

之前：官網+電商 = $5K-10K（客戶看到一個大數字）
現在：基礎官網 $3K + 電商加購 $2K（客戶覺得是自己選的）

心理效果：
├── $3K 看起來不貴 → 先答應
├── 「要不要加電商？+$2K」→ 覺得加一點不多
├── 最後花了 $5K 但覺得「我自己選的，不是被宰」
└── 同樣的價格，客戶接受度完全不同

月費也一樣：
├── 「維護 $249 + 會員 $99 = $348」
├── 比「一個月 $348」聽起來合理
├── 因為客戶知道每一塊花在哪
└── 透明 = 信任
```
